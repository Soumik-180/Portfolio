# Soumik Ray | Portfolio

Welcome to my personal web portfolio! I am a passionate Biotechnology and Bioinformatics student exploring the intersection of biology, data science, programming, and web development to turn complex biological data into meaningful, accessible information.

## Key Features

- **Modern & Minimalist UI**: A clean, light-mode-first aesthetic powered by carefully chosen *Times New Roman* and *Montserrat* typography themes.
- **Dynamic Animations**: Entirely animated via `framer-motion` to create immersive custom physics, from realistic gravity bouncing bubbles in the **Skills** section to a continuous floating smartphone mockup in the **Photography** section.
- **Dark Mode Support**: Seamlessly transitions between a crisp light theme and a sleek dark theme via the fixed glassmorphism navigation bar.
- **Highly Performance Optimized**: Utilizes GPU hardware acceleration, asynchronous image decoding, and `CSS transform` caching to keep the scrolling buttery smooth even with massive graphical elements.
- **Single Page Application (SPA)**: Smooth Anchor-scrolling navigation through React.

## Layout Sections

1. **Hero**: Quick interactive introduction with an animated profile art layout and a continuous looping greeting widget.
2. **About Me**: A customized macOS-style interactive window detailing my education, algorithms, and general background.
3. **Projects**: Detailed case-studies (e.g., Disease Prediction, Protein Sequence Analyzer) showcasing my machine learning and bioinformatics workflow.
4. **Skills**: A fully interactive "Bouncing Physics" container populated with glassmorphic, iridescent technology bubbles.
5. **Education & Achievement**: Symmetrical visual timelines charting my academic timeline, professional milestones, and certifications.
6. **Photography**: A dense 3-column UI gallery featuring 12 custom high-quality images with smooth hover zooms, shadowed by a floating social media phone widget.
7. **Contact**: The final interactive footer containing my animated social links and direct mail configurations.

## Tech Stack

- **Framework**: [React.js](https://reactjs.org/) (initialized via [Vite](https://vitejs.dev/))
- **Styling**: [Tailwind CSS v3](https://tailwindcss.com/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: Lucide React

## Local Installation & Setup

To run this portfolio locally on your machine, follow these detailed steps:

**Prerequisites:**
- Ensure you have [Node.js](https://nodejs.org/) (v16 or higher recommended) installed.
- Ensure you have `npm` installed.

**Installation Steps:**
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/Soumik-180/Portfolio.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Portfolio
   ```
3. Install the required project dependencies:
   ```bash
   npm install
   ```
4. Start the local Vite development server:
   ```bash
   npm run dev
   ```
5. Open your web browser and navigate to `http://localhost:5173` to view the application live. Any changes you make to the code will automatically reflect in the browser.
